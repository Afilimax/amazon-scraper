# @afilimax/amazon-scraper

Este é um projeto para rapar dados de produtos da Amazon focado no marketplace brasileiro. Se acabar funcionando para outros idiomas, é pura sorte. Mas, se você for gringo e quiser contribuir para cobrir o seu ou outro idioma, sinta-se à vontade para abrir um pull request.

## Instalação

```bash
npm install
```
